# Password Generator

A simple and secure password generator built with Python.

## How to Run

Since this is a **Python** project, you should run it using the Python interpreter rather than `npm`.

1. Open your terminal in the project directory.
2. Run the following command:
   ```powershell
   python password_generator.py
   ```

## Features
- Generates strong, random passwords.
- Allows specifying password length.
- Uses the cryptographically secure `secrets` module.
